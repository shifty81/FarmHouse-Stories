### Files Included

All files in the `/gfx/` folder:

1. **Overworld.png** - Main overworld/outdoor tileset (640×576px)
2. **Inner.png** - Interior/building tileset (640×400px)
3. **cave.png** - Cave/dungeon tileset (640×400px)
4. **character.png** - Player character sprite sheet (272×256px)
5. **objects.png** - Objects, items, and tools (528×320px)
6. **NPC_test.png** - Sample NPC character (64×128px)
7. **log.png** - UI dialog box elements (192×128px)
8. **font.png** - Bitmap font for text (240×144px)

### Usage in Project

These assets are used throughout the FarmHouse Stories game:
- Tilesets for game world construction
- Character sprites for player and NPCs
- Objects for farming tools and decorations
- UI elements for dialogue and menus

- Keep color palette limited for cell shading compatibility
- All new sprites should be PNG with alpha transparency
